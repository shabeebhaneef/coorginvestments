# Coorg Investments — Pipeline Setup Guide

## What you're building
Google Form (phone) → Google Sheet → GitHub Action → Site updates automatically

---

## STEP 1 — Rename index.html to index.template.html

In your GitHub repo, rename `index.html` to `index.template.html`.
This becomes the template the generator script uses.

Then find these two lines inside it and replace them:

**Find the property cards div contents and replace with:**
```html
<!-- PROPERTY_CARDS_PLACEHOLDER -->
```

**Find the filter tabs div contents and replace with:**
```html
<!-- FILTER_TABS_PLACEHOLDER -->
```

**Find "Copyright © CoorgInvestments 2025" and add after it:**
```html
<span style="font-size:0.75rem;color:#555;"> · Last updated: <!-- LAST_UPDATED_PLACEHOLDER --></span>
```

## STEP 3 — Link Form to Google Sheet

In the Form → Responses tab → Click the Google Sheets icon → 
"Create a new spreadsheet" → Name it "Coorg Investments Properties"

---

## STEP 4 — Make the Sheet publicly readable (CSV access)

In the Google Sheet:
1. File → Share → Share with others → Change to "Anyone with the link" → Viewer
2. Note your Sheet ID from the URL:
   `https://docs.google.com/spreadsheets/d/YOUR_SHEET_ID/edit`
3. Your CSV URL will be:
   `https://docs.google.com/spreadsheets/d/YOUR_SHEET_ID/export?format=csv&gid=0`

---

## STEP 5 — Add Sheet CSV URL as GitHub Secret

In your GitHub repo:
Settings → Secrets and variables → Actions → New repository secret

- Name:  `SHEET_CSV_URL`
- Value: `https://docs.google.com/spreadsheets/d/YOUR_SHEET_ID/export?format=csv&gid=0`

---

## STEP 6 — Create GitHub Personal Access Token

1. GitHub → Settings (top right avatar) → Developer settings → 
   Personal access tokens → Tokens (classic) → Generate new token
2. Name: "Coorg Investments Deploy"
3. Expiration: No expiration
4. Scopes: Check ✅ `repo` (full control)
5. Generate and COPY the token immediately (shown only once)

---

## STEP 7 — Add Apps Script to Google Sheet

In the Google Sheet:
Extensions → Apps Script → Delete all existing code → Paste contents of `apps-script.gs`

Then:
1. Project Settings (gear icon) → Script Properties → Add property:
   - Key:   `GITHUB_TOKEN`  
   - Value: (paste your token from Step 6)

2. Save the script (Ctrl+S)

3. Add trigger: Clock icon (Triggers) → Add Trigger:
   - Function: `onFormSubmit`
   - Event source: From spreadsheet
   - Event type: On form submit
   - Authorize when prompted

4. Test it: Run → `testTrigger` → Check GitHub Actions tab for a new run

## STEP 8 — Upload files to GitHub repo

Your final repo structure:
```
coorginvestments/
├── index.template.html    ← renamed from index.html
├── index.html             ← auto-generated (don't edit manually)
├── generate.js            ← the generator script
├── properties.json        ← auto-generated by Action
├── properties.csv         ← auto-generated by Action
├── sitemap.xml
├── robots.txt
├── CNAME                  ← auto-created by GitHub Pages
└── .github/
    └── workflows/
        └── update-listings.yml
```

Upload `generate.js` and `.github/workflows/update-listings.yml` to the repo.

## Daily use for your father-in-law

**To add a property:**
Open Google Form link → fill details → upload photos → Submit
→ Site updates in ~3 minutes automatically

**To remove/sell a property:**
Open Google Sheet → find the property row → change Status from `Active` to `Sold`
→ Site updates on next scheduled run (every 6 hours) or trigger manually

**To trigger update immediately after Sheet edit:**
GitHub repo → Actions → Update Property Listings → Run workflow → Run workflow

## Troubleshooting

**Action not triggering?**
- Check Apps Script execution log (Extensions → Apps Script → Executions)
- Verify GITHUB_TOKEN in Script Properties is correct
- Token may have expired — generate a new one

**Photos not showing?**
- Make sure Google Drive folder is shared publicly
- Use the Drive share URL format: `https://drive.google.com/file/d/FILE_ID/view`

**Site not updating?**
- Check GitHub Actions tab for failed runs and read the error log
- Verify SHEET_CSV_URL secret is correct and sheet is publicly readable
